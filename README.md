# User Registration API with MongoDB

A RESTful API built using Node.js, Express.js, MongoDB, and Mongoose for managing user data.

## Features

* Create User
* Get All Users
* Get User by ID
* Update User
* Delete User
* Input Validation
* MongoDB Database Integration

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Installation

```bash
npm install
node server.js
```

## API Endpoints

* GET /users
* GET /users/:id
* POST /users
* PUT /users/:id
* DELETE /users/:id
